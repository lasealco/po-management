# Supply Chain Risk Intelligence — UX / UI and Wireframes

## UX Principles
- show what matters to the customer network, not generic headlines
- emphasize impact and action over article volume
- keep event cards compact but high-signal
- show why the event was matched to internal objects
- distinguish facts from AI interpretation

## Main Screens

### Dashboard
Top cards:
- critical events
- impacted shipments
- impacted POs
- impacted orders
- impacted suppliers
- active recommendations

### Event Feed
Card layout:
- headline
- 2-line summary
- event type
- geography
- severity
- confidence
- impacted object counts
- actions

### Event Detail
Sections:
- event summary
- source panel
- impact panel
- affected objects tabs
- recommendations
- triage log

### Triage Workspace
Actions:
- assign owner
- mark watch
- dismiss
- create task
- launch twin scenario
- notify stakeholders

### Watchlist Center
Configure watch rules and preferences.
