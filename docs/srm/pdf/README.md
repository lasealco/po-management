# SRM PDF pack

This directory holds the **versioned, human-readable** SRM specification PDFs (dated filenames, e.g. `_20260417_063215`).

## Rules

- **Do not** add duplicate `*.pdf` files under `docs/srm/` (repo root of SRM docs). PDFs belong **here only**.
- For agents and IDE search, use the Markdown sidecars under **`../md/`** (same topic, shorter basename — see **`../README.md`** pairing table).
