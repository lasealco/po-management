- category
- status
- region
- risk
- score
- strategic tier
- approved services
- contract state
- document health

- supplier code
- legal name
- trading name

- status
- strategic tier
- owner
- region

- performance score

## 2. Process Screens

### Supplier Application Intake
Capture new supplier request or application manually, via portal, or from internal nomination.

### Onboarding Checklist
Track all required steps before supplier can move to approved/active status.

### Document Request Center
Request missing documents and track submission/approval.

### Qualification Questionnaire
Structured assessment by supplier type or service category.

### Review Workspace
Annual/quarterly review preparation and scoring.

### Issue / CAPA Workspace
Corrective and preventive actions against incidents or audit findings.
